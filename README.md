# 사이트 개요

- 앱 이름 : F.O.C(Fast Of Calcultaion)

- 주요 기능 : 암산 게임 앱으로, 각 게임 파트(덧셈, 뺄셈, 곱셈, 나눗셈, 혼합계산)별 10문제를 풀고 기록을 재는 게임

# 목적성

- 암산 게임으로 기록을 재면서 수학적 계산 능력의 증가

- 5개의 연산 파트(덧셈, 뺄셈, 곱셈, 나눗셈, 혼합계산)별로 암산 게임을 하며 복합적인 계산능력의 증가 및 뇌 활동량 증가
